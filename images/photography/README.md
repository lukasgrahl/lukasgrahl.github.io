# Adding a photograph

1. Copy the image into this folder using a short lowercase filename, for example `aarhus-harbour.jpg`.
2. Open `photography.qmd`.
3. Copy one existing `<figure>...</figure>` block.
4. Change the filename, alternative text, and caption.
5. Run `quarto preview` to check the page before committing.

The photographs in this folder are personal work by Lukas A. Grahl and are not provided under an open-content licence.
