# Downloadable files

Place public PDFs and other downloads in this folder.

## CV

1. Save the current PDF as `files/cv.pdf`.
2. Open `cv.qmd`.
3. Uncomment the download-button line near the top of the file.

## Research papers

Save papers, appendices, and slides in `files/papers/`. See `files/papers/README.md` for the naming and linking pattern.
